96  // playout capabilities
97  // (http://dashif.org/wp-content/uploads/2016/12/DASH-IF-IOP-v4.0-clean.pdf,
98  // page 18, line 1). It is the ratio of main frame rate to the trick play
99  // frame rate. If the time scale and frame duration are not modified after
100  // trick play handler processing, the playback_rate equals to the number of
101  // frames between consecutive key frames selected for trick play stream. For
102  // example, if the video stream has GOP size of 10 and the trick play factor
103  // is 3, the key frames are in this trick play stream are [frame_0, frame_30,
